gtitlebar (标题管理器) 

        不支持HTML定义,需要用script 定义，主要用于被继承。
     
        <div id="qq" style="width:400px;" ></div>
        $("#qq").gtitlebar({collapse:true,header:"33333333333333333",headerIcon:"add"});
Properties(属性)：
Name Type Description Default
max Boolean 是否显示最大化按钮 false
collapse Boolean 是否显示展开/收起按钮(上下) false
collapse1 Boolean 是否显示展开/收起按钮(左右) false
close boolean 是否显示关闭按钮 true
border Array 显示边框[1,0,1,1],为0则不显示
header string 标题
fit Boolean 初始化显示最大或最小图标，max 为false 无效
headerIcon string 标题左边的图标
