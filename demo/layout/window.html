gwindow (窗口管理器) 
    <input type="button" onclick="showwindow(1)" value="显示窗口" />
    <div id="mywindow" class="gwindow" data-options="{headericon:'add',header:'我的窗口',drageable:true,resizeable:true,shadowable:true,mask:true}" style="width:400px;" >
        我是窗口
    </div>
        //script:
         function showwindow(fg) {
            jgui.get('mywindow').open();
        }
Properties(属性)：
Name Type Description Default
header string 窗口标题内容
headerIcon string 显示在标题栏左边的图标
mask Boolean 是否模态窗口 false
drageable Boolean 是否可拖动 true
resizeable Boolean 是否可改变大小 false
collapse Boolean 是否显示展开、收起按钮 false
isHeightFit Boolean 是否自适应高度 false
isOverflow Boolean 超出窗口边界时是否溢出，有date或combo控件时应设为false,允许超出窗口边界 true
shadowable Boolean 是否显示阴影 true
closed Boolean 生成后是否先隐藏 true
close Boolean 是否显示关闭按钮 true
width Number 宽度 300
height Number 高度,当isHeightFit为true时无效 200
max boolean 是否显示最大化按钮 false
Methods(方法)：
Name Parameter Description Return
open 显示窗口
close 隐藏窗口
destroy 删除窗口
