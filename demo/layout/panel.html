gpanel (面版管理器) 
     <div class="gpanel" style="width:400px;" >
        我是panel 
    </div>
Properties(属性)：
Name Type Description Default
header string 标题内容
headerIcon string 标题左边图标
fit boolean 是否最大化 false
width Number 宽度 300
height Number 高度 200
close boolean 是否显示关闭按钮 true
collapse boolean 是否显示展开/收起按钮 true
max boolean 是否显示最大化按钮 true
isOverflow boolean 溢出时是否显示滚动条 true
