glayout (布局管理器) 
    <div id="layout1" class="glayout" data-options="{fit:true,splitSize:3,width:400,height:300}" >  
        <div data-options="{region:'top',height:60,title:'1111111',collapse:true,close:true,border:[1,1,1,1]}" >
            顶部
        </div>  
        <div data-options="{region:'bottom',height:68,border:[1,1,1,1],title:'44444',close:true,collapse:true}" >
            底部
        </div>  
        <div data-options="{region:'right',icon:'folder',title:'右边',width:100,border:[1,1,1,1],collapse:true}">
         中部
        </div>  
        <div data-options="{region:'left',title:'左边',width:200,border:[1,1,1,1],collapse:true,}" >
            左边
        </div>  
        <div data-options="{region:'center',border:[1,1,1,1]}" >
            中部
        </div>  
    </div>
Properties(属性)：
Name Type Description Default
fit Boolean 是否最大化 true
width Number 宽度 400
height Number 高度 300
splitSize Number 分隔宽度(像素) 2
Methods(方法)：
Name Parameter Description Return
refresh 重新布局
reseth 重新布局高度
Events(事件):
Name Parameter Description Return
Region Object(面板对象)
Name Type Description Default
region String 布局，top,bottom,left,giht,center之一
width Number 宽度,top 和bottom，center 没有该属性
height Number 高度,left.center,right没有该属性
title String 是否显示标题 null
collapse String 是否显示展开、收起按钮 null
close Boolean 是否显示关闭按钮 true
headerIcon String 标题的图标
