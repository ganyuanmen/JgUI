gtext (文本框) 

     <div class="gtext"  data-options="{name:'t1',value:23}"  > </div>
    或者
     <div class="gtext" name="t1"  data-options="{value:23}"  > </div>
Properties(属性)：
Name Type Description Default
name string 对象名称，在form的getData中用到
width Number 宽度 100
value string或Number 值
typeValid string 数据类型 int 或float
valid function 验证函数 
                        function comvalid(v) {
                            if (v > 100) {
                                this.validMessage = '不能大于100';
                                return false;
                            } else
                            return true;
                         }
Methods(方法)：
Name Parameter Description Return
setValue value 赋值
getValue 取值
Events(事件):
Name Parameter Description Return
onChange function 值改变时确发
