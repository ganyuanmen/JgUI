gtree (树形控件) 

    <ul class="gtree" data-options='{url: "data/menutree.txt",click:showmenu}'></ul>
Properties(属性)：
Name Type Description Default
url string 数据源url
para object url的参数
click function 单击时的callback
clickTarget obj click的this 对象 null
checkboxNode Boolean 带checkbox false
width Number 宽度 200
height Number 高度 300
expandOnClickItem Boolean 单击时是否展开下级菜单 false
fit Boolean 自动最大化 true
>tree 项目Properties(属性)：
Name Type Description Default
id string 项目ID id
pid string 父ID pid
text string 显示文本
expand string 初始化时是否打开"true"或"false"
url string
