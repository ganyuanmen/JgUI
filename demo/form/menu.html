gmenu (下拉菜单) 

     HTML5 定义有一套完整的menu组件，目前的浏览器似乎未支持，此处只定义单击下拉框。
     <a class="gmenu" data-options="{itemClick:iclick,poupuid:'#itemid'}">更多...</a>

    <div id="itemid" style="display:none;">
          <div data-options="{icon:'reload',fa:4}">打印</div>
    </div>
Properties(属性)：
Name Type Description Default
itemClick function 单击触发click
poupuid string 菜单内容
Options项(属性)：
Name Type Description Default
icon string 菜单项图标
.... object 定义其它参数，在itemclick中可调用
