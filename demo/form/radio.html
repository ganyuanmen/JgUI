gradio (单选按钮) 

     <div class="gradio" data-options="{name:'sex',width:110,data:[{id:true,text:'男'},{id:false,text:'女'}]}" > </div>
Properties(属性)：
Name Type Description Default
value id的值 初始化值，无此值则第一项为默认值
width number 宽度
data array 单选项{id:'id',text:'text'}
align sting 对齐方式:left,right,center
