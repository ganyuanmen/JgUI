gfile (文件选择、上传按钮) 

    <a class="gfile"  data-options="{icon:'add',fileChange,fileLoad:loadd}">选择文件</a>
Properties(属性)：
Name Type Description Default
fileChange function 选择文件后触发,用于检测文件类型 ，返回true时才能触发fileLoad 反回true
icon string 按钮图标
fileLoad function 文件加载成功后触发，此处使用本地文件读取
