gcombo (下拉选择框) 

     <div class="gcombo"  data-options="{data:[{id:1,text:'1111111'},{id:2,text:'2222222222'}]}"  > </div>
Properties(属性)：
Name Type Description Default
name string 对象名称，在form的getData中用到
width Number 宽度 200
valueId string getValue时取值字段 id
id string 数据源的id id
text string 数据源的文本,选择时显示的内容 text
data Array 数据源
value string 初始值
url string 数据源的url
poupuHeight Number 弹出框的高度 200
poupuWidth string 弹出框的宽度 200
isup boolean 弹出框是否在选择框上面 false
islockheight boolean 是否固定高度 false
edit boolean 是否允许键盘修改 false
isFilter boolean 是否启用过滤 false
valid function 验证函数 
                        function comvalid(v) {
                            if (!v ||v > 1) {
                                this.validMessage = '只能选择第一个';
                                return false;
                            } else
                            return true;
                         }
Methods(方法)：
Name Parameter Description Return
setIndex Number 根据数据源的序号设定combo的值
setValue string 设定combo的ID
setData Array 设数据源
disable Bollean true 为不活动状态, false 活动状态
setUrl string 设数据源的url
getValue 获值 string
getText 获取选择中的文本 string
getrecord 获选中的数据记录 object
getIndexFromId Number 通过ID获数据源中的序号 Number
valid function 验证
Events(事件):
Name Parameter Description Return
onChange function 值改变时确发
